**Coursera Universal Subtitles** is a powerful Chrome Extension designed to break the language barrier on Coursera. It overlays translated subtitles directly onto the video player, allowing for a seamless and immersive learning experience.

## ðŸŒŸ Key Features
- **Draggable Overlay:** Displays active subtitles in the center-bottom of the video player and can move the subtitles anywhere
- **Translation Support:** Works seamlessly with Google Chrome's built-in "Translate" feature.
- **Smart Sync:** Automatically detects and displays the current phrase being spoken.
- **Clean UI:** High-contrast text with a subtle background for readability on any video content.
- **Visual Customization:** Adjust font size and background opacity directly from the extension popup.

## ðŸ“– How to Use
1. **Open a Course:** Start any video on Coursera.
2. **Translate:** Right-click the page and select **"Translate to [Your Language]"**.
3. The extension will automatically pick up the translated text and show it on the video.
4. **Customize:** Click the extension icon (blue logo) to:
   - Change **Font Size** if the text is too small/large.
   - Adjust **Opacity** to make the background more or less transparent.
5. **Move:** Click and drag the subtitles to your desired position.
